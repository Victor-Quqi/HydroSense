本任务旨在编写 ULP（哨兵）协处理器上运行的核心代码，实现低功耗的周期性环境感知。

-   **任务清单:**
    1.  在 `ulp/ulp_main.c` 中编写 C 代码。
    2.  实现逻辑：打开传感器电源 -> 延时等待稳定 -> 读取土壤湿度 ADC 原始值 -> 关闭传感器电源。
    3.  **必须使用定点整数**进行所有操作，严禁使用浮点数。
    4.  将读取到的 ADC 原始值存储在一个 `volatile` 全局变量中，以便主 CPU 读取。
    5.  定义一个 `volatile` 全局变量 `ulp_run_counter`，每次 ULP 运行时自增，用于调试。
    6.  如果湿度低于预设阈值（硬编码），则触发一个 `WAKE_MAIN_CPU` 的动作。

-   **验收标准:**
    1.  在测试模式下添加 CLI 命令 `ulp read`，该命令读取并打印 `ulp_run_counter` 和湿度传感器的共享变量值。
    2.  手动触发 ULP 运行后，通过 `ulp read` 命令可以确认 `ulp_run_counter` 增加，并且能获取到有效的 ADC 读数。
