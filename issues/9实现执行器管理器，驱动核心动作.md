本任务旨在激活 `actuator_manager`，使其能够精确控制水泵和电磁阀。它将利用ESP32的LEDC（PWM）功能实现水泵功率调节，并协同 `power_manager` 和 `hal_gpio` 来完成高功率外设的驱动。

-   **核心需求:**
    1.  **电磁阀控制:** 实现对电磁阀的开关控制，作为防止滴漏的双重保险。
    2.  **水泵PWM功率控制:** 实现对水泵运行速度的调节，以适应不同植物和浇水场景。
    3.  **协同安全操作:** 任何执行器动作都必须先开启12V电源，动作结束后再关闭电源。

-   **任务清单:**
    1.  **HAL层增强:**
        *   创建 `hal_ledc.h/cpp` 模块，封装ESP32的LEDC外设。
        *   提供接口如 `hal_ledc_init(pin, channel)` 和 `hal_ledc_set_duty(channel, duty_cycle)`。
    2.  **管理器接口扩展:**
        *   在 `managers/actuator_manager.h` 中定义新的业务接口：
            *   `actuator_manager_set_valve(bool is_open)`
            *   `actuator_manager_run_pump(uint8_t duty_cycle, uint32_t duration_ms)`
    3.  **管理器实现:**
        *   在 `managers/actuator_manager.cpp` 中实现新接口。
        *   `set_valve` 逻辑: 打开12V电源 -> 设置GPIO电平 -> 关闭12V电源。
        *   `run_pump` 逻辑: 打开12V电源 -> 设置PWM占空比 -> 等待时长 -> PWM占空比归零 -> 关闭12V电源。
        *   **注意:** 需考虑两个执行器同时或连续动作的场景，避免频繁开关12V电源。管理器内部应维护一个电源状态标记或引用计数。

-   **验收标准:**
    1.  在 `test_commands_hal` 中添加或修改CLI命令：
        *   `valve <on|off>`
        *   `pump run <duty_cycle> <duration_ms>` (duty_cycle: 0-255)
    2.  执行 `valve on` 后，电磁阀打开；`valve off` 后关闭。
    3.  执行 `pump run 128 2000` 后，水泵以约50%功率运行2秒。
    4.  在所有操作期间，12V电源都按预期被正确管理。
